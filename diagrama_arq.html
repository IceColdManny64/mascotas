Presentación
Jinja2 Bootstrap 5.3 JS Vanilla Fetch API
  • Renderizado server-side completo con Jinja2

  • Herencia de layouts con {% extends %}

Aplicación — Flask 3.x
Application Factory Flask-Login Flask-WTF CSRF @roles_required
auth_bp / /login /register Registro, login, logout. Gestión de sesiones con Flask-Login.
pets_bp /pets CRUD de mascotas, búsqueda con filtros, gestión de fotos.
adoptions_bp /adoptions /shelter Solicitudes de adopción, favoritos, panel del refugio.
alerts_bp /alerts Configuración de alertas de búsqueda personalizadas.
messages_bp /messages Mensajería interna entre adoptante y refugio.
reviews_bp /shelters /reviews Perfiles de refugio, valoraciones y reportes.
admin_bp /admin Dashboard de métricas y herramientas de moderación.
Autenticación
  • Sesiones gestionadas por Flask-Login
  • Contraseñas hasheadas con Werkzeug
  • Roles: adopter · shelter · admin
Autorización
  • Decorador @roles_required por ruta
  • Suspensión de cuentas (is_suspended)
Protección
  • CSRF en todos los forms POST vía Flask-WTF
  • Validación de formularios con WTForms
